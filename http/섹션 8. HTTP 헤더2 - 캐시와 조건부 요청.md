## 캐시 시간 초과
- 캐시 유효 시간 초과하면, 서버를 통해 데이터를 다시 조회 후 갱신함
- 네트워크 다운로드 다시 발생
- 클라이언트의 데이터와 서버의 데이터가 같다는 사실을 확인할 수 있으면 다운로드 안 받아도 됨
